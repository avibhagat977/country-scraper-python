# Country Data Scraper & Analyzer

This project is a complete python data pipeline that scrapes country information from
http://www.scrapethissite.com/pages/simple/ and performs data storage, analysis and visualization.

It demonstrates real-world skills in:
- Web Scraping
- Data Cleaning 
- SQLite Database
- Pandas Analysis
- Matplotlib Visualization
- Logging and Error Handling
- Modular Python Project Structure

# Features

- Scrapes country name, capital, population and area
- Saves data to CSV
- Stores data in SQLite database
- Analyzes data using pandas
- Generates charts using matplotlib
- logs all operations and errors
- Fully automated with 'main.py'

# Technologies Used

- Python
- Requests
- BeautifulSoup
- SQLite3
- Pandas
- Matplotlib
- Logging
